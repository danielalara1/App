# Organización 

Para la gestión de este proyecto se ha implementado la metodología "Kanban" mediante la herramienta "Trello". Esra elección permite una visualización clara del flujo de trabajo.

## Estructura

El tablero se divide en cinco columnas principales: 

* Blacklog: Ideas y funcionalidades secundarias.
* Todo: Tareas priorizdas.
* In Progress: Tareas en desarrollo.
* Review: Funcionalidades terminadas pendientes de testing.
* Done: Treas completadas,
