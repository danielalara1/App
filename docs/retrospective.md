# Retrospectiva y Reflexión Final

## Aprendizajes
Este proyecto ha sido fundamental para entender la conexión real entre un Frontend (React) y un Backend (Express). He aprendido a estructurar una aplicación por capas y a gestionar la persistencia de datos en MongoDB.

## Desafíos
El mayor reto fue la configuración del entorno y la conexión de red entre el cliente y el servidor. Surgieron dificultades con la base de datos en la nube, que se resolvieron utilizando un entorno local con MongoDB Compas y exponiendo los puertos de forma segura.

## Uso de IA
Durante el desarrollo, he utilizado la IA como asistente para:
*   Depurar errores de conexión entre el puerto del servidor y el cliente.
*   Optimizar la estructura de los archivos de documentación.
*   Generar interfaces de TypeScript alineadas con el esquema de la base de datos.
La IA ha servido como un tutor para explicar conceptos técnicos y acelerar el proceso de documentación.
