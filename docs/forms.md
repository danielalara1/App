# Formularios e Interacción

La aplicación cuenta con un formulario de creación de "Vibes" que gestiona la interacción del usuario de forma segura.

## Características
*   Inputs Controlados: Cada campo del formulario está vinculado a un estado de React .
*   Validación: Se comprueba que los campos obligatorios  no estén vacíos antes de enviar la petición.
*   Feedback: El usuario recibe un mensaje de confirmación o error dependiendo del resultado de la petición POST al backend.
