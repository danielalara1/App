#  Batnie Project - Social Vibe App

Este proyecto es una aplicación web full-stack diseñada para compartir "Vibes" (publicaciones con imagen y descripción). Desarrollada como parte de la Fase 4 de la formación, enfocándose en la arquitectura modular y la integración de sistemas.

##  Tecnologías Utilizadas

### Frontend
- Framework: React con Vite y TypeScript.
- Estilos: Tailwind CSS para un diseño responsivo y moderno.
- Enrutamiento: React Router para la navegación entre páginas.

### Backend
- Entorno: Node.js con Express.
- Base de Datos: MongoDB, gestión local a través de MongoDB Compass.
- Comunicación: Axios para peticiones HTTP al servidor.
